# Elevated Health Augusta

Physician-owned and operated wellness clinic in Evans, GA. This is the patient-facing website plus internal provider/admin/patient portal, built on Vite + React + TypeScript + shadcn-ui + Tailwind + Supabase.

## Stack
- Vite + React + TypeScript
- shadcn-ui + Tailwind CSS
- Supabase (auth, database, edge functions, storage)
- Stripe (payments, subscriptions)
- LabCorp + ZRT integrations
- GHL CRM hand-off

## Local development
```bash
npm install
npm run dev
```

## Brand
Bone (#F2EBDC), Warm Charcoal (#2A2826), Camel (#B8956A). Playfair Display for headlines, Jost for UI/body.
